## ‚öôÔ∏è Running the Application

Once you have installed AI Home Guardian, follow these steps to start:

1. Open the application on your device.
2. Connect your webcam or mobile camera.
3. Create a Telegram bot using the Telegram app, following these steps:
   - Search for the BotFather in Telegram.
   - Send the command `/newbot` and follow the prompts to create a new bot.
   - Save the Bot Token provided by BotFather.
4. In the application settings, enter your Bot Token to connect AI Home Guardian with Telegram.
5. Choose the area you want to monitor and set up motion detection settings.
6. Start monitoring your space. You will receive alerts directly to your Telegram whenever motion is detected.
